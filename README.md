# TL_DV_Demo
A full stack application built with Strapi and React for viewing and editing video 

## Installation

Frontend

```bash
cd frontend 

yarn install

yarn start
```



Backend

```bash
cd backend 

yarn install

yarn develop
```

Visit http://localhost:3000
